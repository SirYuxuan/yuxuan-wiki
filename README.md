# 雨轩 Wiki

这是一个使用 Docusaurus 2 构建的个人知识库网站，用于记录学习与生活。

## 特点

- 🚀 基于 Docusaurus 2 构建
- 📝 支持 Markdown 文档
- 🎨 美观的界面设计
- 🌙 支持深色模式
- 🔍 全文搜索功能
- 📱 响应式设计

## 开发

```bash
# 安装依赖
npm install

# 本地开发
npm run start

# 构建
npm run build

# 部署
npm run serve
```

## 文档结构

- `/docs/rust/` - Rust 相关文档
- 更多文档正在添加中...

## 技术栈

- [Docusaurus 2](https://docusaurus.io/)
- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
